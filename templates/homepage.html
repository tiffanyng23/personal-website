Spotify Artist Explore Page

A reimagined Spotify artist page with the aim of encouraging a more thorough and album-driven exploration of an artist's discography. The artist page displays album tracks based on Spotify popularity trends, with a focus on identifying "mainstream" and "hipster" tracks. Albums are displayed in a table with information including popularity score and album length, making it easier for users to view key album information on one page.

Earthquake Tracker

A global earthquake tracker that integrates the USGS earthquake api to provide live updates on earthquake events. Features an earthquake counter, scatter map, scatter plot, and data table that are customizable by time window, depth, and magnitude of earthquakes to allow for user-driven analysis of earthquakes.

NASA Image Repository

A website that uses the NASA Open API to display recent images of Earth taken from the EPIC camera in space, and NASA's Astronomy Picture of the Day (APOD) - daily photographs released by NASA. This website also features a map displaying wildfire, severe storm, and volcanic eruption events.
