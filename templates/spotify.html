About this Project

This project is my reimagined version of the artist page on Spotify using Spotify's Web API. My goal with this project was to build the Spotify Artist Page in a way that focuses on showcasing albums and the album's tracks. I generally prefer to explore artist's through their albums since I feel that their albums show their artistic vision compared to non-album tracks. I also prefer album-based listening over playlists because I like concept albums where I can notice overarching sounds or themes across each track. I really feel that through this project I've been able to create an artist page that combines an album-based listening approach with displaying tracks dynamically based on Spotify popularity value (scored from 0-100, based on number of streams and recency of streams), which I hope will encourage deeper exploration of an artist's discography.

Albums Tab

The Albums Tab displays all the artists albums in a table, which allows for users to search, filter, and sort album information.

Album Tracks Tab

The Album Tracks Tab displays all the tracks from artist albums (not including non-album tracks). There are 3 lists sorted based on the Spotify popularity score. The "Most Mainstream Album Tracks" are the 10 tracks of the highest popularity value, providing tracks that are trending and are likely fan favourites. The "Most Hipster Album Tracks" are the 10 tracks with the lowest popularity value, suitable for finding hidden gems. The middle list allows for users to create a customized list based on the Hipster vs. Mainstream ratio, providing a list of 5 tracks that leans more "mainstream" or more "hipster".

The bottom portion of this tab provides a bar chart displaying the artist's album tracks based on its popularity value, in either ascending or descending order.

Still Not Sure Where to Start Tab

This tab is perfect for users who simply want to dive into any album or track by the artist. This tab features the Spotify Generated Top Tracks which are the 10 featured tracks found on the top of an artist page on the Spotify app. This will differ from the "Most Mainstream Album Tracks" list since it also includes non-album tracks and tracks where the artist is a feature. This tab also has two shuffle-like buttons: a random album generator and random album track generator. By clicking on the button, the user will be redirected to a randomnly selected album or track respectively.
